# DimCatcher
A Python-based OCR tool that automatically extracts, tallies, and sorts shipping box dimensions from daily ShipStation packing slip PDFs.
How to Use DimCatcher

Open the Tool: Click your bookmarked DimCatcher link to open the web app.

Upload the Labels: Drag and drop your daily ShipStation PDF into the upload box on the screen (or click "Browse Files" to select it).

Wait for the Scan: The app will display a loading spinner while it reads the pages.

Download the Tally: Once finished, click the "Download Box Tally (PDF)" button that appears.

Print: Open the downloaded PDF and hit print.

Reset: To run another file, just hit the "X" on your uploaded file in the drag-and-drop box, or simply refresh the webpage!
